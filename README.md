# Introduction à la Programmation Système

## Dépendances

- **`fichiers_td` :** Les dépendances nécessaires pour compiler les programmes sont : `gcc`, `flex`, `bison` et la librairie `readline`.
- **Fichiers `.typ` :** Pour les fichiers `.typ` (dans `enonce_corr_td`), il faut installer `typst` et éventuellement `touying`.

## Commandes utiles

- **`fichiers_td` :** On peut utiliser `make` dans le dossier `fichiers_td` pour lancer la compilation des exercices.
- **Fichiers `.typ` :** On compile un fichier Typst avec la commande : `typst compile <fichier>`. Pour obtenir un fichier HTML (support des GIFs), on utilise `touying compile <fichier>`.
